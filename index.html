RULES FOR FLIPFLOPS
* To use the flipflop, first give the input number (strictly 0 or 1 only) in the input box near the "Set" text in the flipflop.
* After giving the input, click the clock button (CLK) which is either blue or green in color near the flipflop.
* At once you click the clock button, the output result will be displayed.
* Ensure the output with the table given below.
* To reset all the inputs, click the reset button (small, red color circle-shaped button placed on the body of the flipflop).
* Giving no inputs and non-binary inputs throws an error in all flipflops
* This rule is common for all the flipflops.
RULES FOR LOGIC GATES
* Click on any of the listed blue tabs of the desired logic gate you want to simulate.
* Click on the checkboxes placed on the body of the logic gates (either one of the two checkboxes or both).
* The result will be shown dynamically near the logic gate.
* Even if no checkboxes are selected some negative logic gates (NOT gate, NAND gate) show the output as 1 based on the digital logic.
* Click on the blue tabs or the checkbox(s) once to simulate or select and twice to deselect.
* Giving no inputs are allowed in logic gates.
GENERAL RULES
* Simulation, logic diagram and truth table is given for all the digital circuit simulations for user reference.
* There is no separate back button for this website. Use browser buttons or functions for navigation.
* In the presence of any notification boxes or alert messages, read the message and kindly click "OK" button to pass them.
* Please read all the instructions for efficient usage of this website
